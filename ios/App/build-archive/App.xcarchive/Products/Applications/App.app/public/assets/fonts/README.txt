[한줄 다짐 위젯 폰트 폴더]

이 폴더에 원하는 폰트 파일을 넣으면 "한줄 다짐" 위젯에 자동 적용됩니다.

● 파일 이름을 아래 중 하나로 맞춰주세요 (확장자 그대로):
   - motto.woff2   (가장 권장 - 가볍고 빠름)
   - motto.otf
   - motto.ttf

  예) 내려받은 폰트가 "Cafe24Ssurround.ttf" 라면
      → motto.ttf 로 이름만 바꿔서 이 폴더에 넣기

● 여러 개 넣어도 됩니다(브라우저가 지원되는 것 1개 자동 선택).
● 파일명을 다르게 쓰고 싶으면 알려주세요 — css/style.css 의
  @font-face(MottoFont) 경로를 맞춰드리겠습니다.
