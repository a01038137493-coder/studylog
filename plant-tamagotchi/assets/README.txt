========================================
정원 — 장면 이미지 넣는 법
========================================

이 폴더에 장면 이미지를 넣으면 게임 화면에 그대로 표시됩니다.

[필수]
  파일 이름:  scene.png
  위치:        plant-tamagotchi/assets/scene.png

즉, 당신이 가진 스타듀밸리풍 레퍼런스 이미지를
  scene.png
라는 이름으로 이 폴더에 저장하면 됩니다.

저장 후 브라우저에서 index.html 을 새로고침(F5) 하세요.

------------------------------------------
권장 사양
------------------------------------------
- 형식: PNG (또는 jpg 도 가능하나 png 권장)
- 픽셀아트 원본 해상도 그대로 (작아도 됩니다 — 화면에서
  또렷하게 확대되도록 image-rendering: pixelated 적용됨)
- 가로:세로 비율은 자유 (이미지 비율 그대로 표시됨)

------------------------------------------
참고
------------------------------------------
- 이미지에 이미 "Day 18 / 햇님" 같은 HUD가 그려져 있으면
  그대로 두세요. (기본값)
- 만약 배경에 HUD가 없어서 위에 나무 팻말을 덧씌우고 싶다면
  script.js 의   const SHOW_HUD = false;   를 true 로 바꾸세요.
- 다른 이름을 쓰고 싶으면 script.js 의 SCENE_SRC 를 수정하세요.
